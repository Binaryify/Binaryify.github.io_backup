3D立方体的实现

CSS的transform配合keyframes和animation很强大,可以做很多动画,而且transfrom有个transform-style的属性,可以用来设置3D,做一个3D的动画

transform-style: preserve-3d;

比如可以做个3D立方体

demo: 戳我

HTML结构很简单,就一个ul包住6个li,CSS方面是给ul设置-webkit-transform-style: preserve-3d,让动画变成3D形式的,再给6个li分别设置不同的transform,让这6个li成为个立方体,然后在给ul设置旋转的动画,然后一个旋转的立方体就出现了,还可以修饰下,给不同的li加颜色变化的动画
同理,我们也可以做一个3D的视频播放器什么的,这里就不详细介绍了,因为原理都是一样的

2015-06-18

闭包加法器小程序

闭包是指有权访问另一个函数作用域中的变量的函数，创建闭包的最常见的方式就是在一个函数内创建另一个函数，通过另一个函数访问这个函数的局部变量。使用闭包有一个优点，也是它的缺点，就是可以把局部变量驻留在内存中，可以避免使用全局变量。全局变量在每个模块都可调用，这势必将是灾难性的。（所以推荐使用私有的，封装的局部变量。）一般函数执行完毕后，局部活动对象就被销毁，内存中仅仅保存全局作用域。但闭包的情况不同！

简单来说闭包拥有三个特性：

1.函数嵌套函数
2.函数内部可以引用外部的参数和变量
3.参数和变量不会被垃圾回收机制回收

简单实现jQuery 中的选择器

简单demo,核心代码:

    <script type="text/javascript">

    var $ = function(selector) {
return document.querySelector(selector);
}

    var $$ = function(selector) {

                return document.querySelectorAll(selector);

            }

    $('#div').addEventListener('click',function(){

    alert('1');

    },false);

    $$('div')[0].onclick=function(){

    alert('2');

    }

    </script>

不是任何时候都适合用jQuery的,比如移动端,通过这段代码就可以实现jQuery的选择器了(当然jQuey的实现比这复杂),大大减少代码量,如果不要求兼容性的话够用了(兼容IE8+,Chrome,firefox)
