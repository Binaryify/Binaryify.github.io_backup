ajax加载动画的实现

网上有很多不错的css和svg加载动画,但是如何使用它们呢?下面是使用AJAX技术是用这些动画的源码,原理是在动画页面使用AJAX加载另外一个要加载的页面,并且为了保证动画能持续一段时间而防止加载的页面小导致加载动画一闪而过使用了setTimeout这个方法
JS:


    function haha(){
        $(‘body’).load(‘index.php’,function(a,status,c){
            // console.log(status);
            if(status==’error’){
                $(body).text(‘加载错误’);
            }
        })
    }
    setTimeout(“haha()”,2000);

需要调用jquery或者zepto
