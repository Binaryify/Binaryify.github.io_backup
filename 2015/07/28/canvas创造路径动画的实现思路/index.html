canvas创造路径动画的实现思路

333

众所周知,svg可以用来做路径动画,我在之前的一篇文章也提过,(demo)[http://zhuangtongfa.github.io/2015/05/25/SVG%E6%8F%8F%E8%BE%B9%E5%8A%A8%E7%94%BB%E5%AE%9E%E7%8E%B0/]
但是canvas这种一次画完的的怎么实现路径动画呢?
下面是实现canvas路径动画的思路
demo
原理很简单,用定时器逐帧画出,并且每次画都是在原来的基础上画,然后画完的时候清除计时器即可
核心代码:

function drawCircle(x, y, r) {

ctx.strokeStyle = "red";

var i = 1;

if (i < 100) {

  var interval = setInterval(function() {

    ctx.beginPath()

    ctx.arc(x, y, r, 2 * Math.PI * (i - 1) / 100, 2 * Math.PI * i / 100);

    ctx.stroke()

    console.log(i / 100)

    i++;

    if (i > 100) {

                  clearInterval(interval);

    }

  }, 10)

}

    }
