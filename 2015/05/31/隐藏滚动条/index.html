隐藏滚动条

  • HTML5
  • css

滚动条也是伪元素,所以也可以用css改变它的样式


    ::-webkit-scrollbar{
        width:10px;
        background-color:red;
    }

也可让width为0,让滚动条隐藏,可以配合overflow:hidder做一个tab菜单
