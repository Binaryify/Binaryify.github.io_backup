div宽度按屏幕比例,让高度自适应和宽度相同非js解决方案

  • css

有时候我们给div宽度设为百分比,但是同时又想高度也跟宽度一样,怎么办呢?其实可以利用伪元素和padding-bottom属性来解决

    div{

            width: 20%;

            background-color: green;

            display: inline-block;

        }

        div:before{

            content:"";

            padding-bottom: 100%;

            display: inline-block;

        }

主要用到了padding-bottom 这个属性
设置padding-bottom值来计算div的纵横比例。如果宽高比是4:3，设置padding-bottom值为75%；如果是1:1,则为100%;
